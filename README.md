# Xiaolan-Home

纯静态主页，功能十分简单。仅需修改config.js即可完成对网站的自定义。

---

# Config.js配置内容

## 顶层字段

```js
const SITE_CONFIG = { title, favicon, name, tagline, avatar, background, footer, sections }
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `title` | string | 浏览器标签页标题 |
| `favicon` | string | 标签页图标 URL |
| `name` | string | **必填**。页面中央显示的名字 |
| `tagline` | string | **必填**。副标题/签名 |
| `avatar` | string | 头像图片 URL |
| `background` | string | 全屏背景图片 URL |
| `footer` | string | 页脚文字，支持 HTML（如 `<span class="heart">❤</span>`） |
| `sections` | array | 内容板块数组，按顺序渲染 |

---

## sections 板块类型

### `contact` — 联系方式

以网格卡片展示，有 `link` 则跳转，无 `link`（设 `null`）则点击复制 `value`。

| 字段 | 必填 | 说明 |
|------|------|------|
| `label` | 否 | 小号灰色标签文字 |
| `value` | **是** | 主要内容（邮箱、ID 等） |
| `link` | 否 | 跳转链接；`null` = 点击复制 |
| `icon` | 否 | Emoji / 文字 / 图片 URL（以 http 开头自动转 `<img>`） |
| `iconSvg` | 否 | SVG 字符串，优先级高于 `icon` |

```js
{ type: 'contact', title: '联系方式', items: [
  { icon: '✉', label: 'Email', value: 'hi@example.com', link: 'mailto:hi@example.com' },
  { icon: '微', label: 'WeChat', value: 'my_wechat_id', link: null }   // 点击复制
] }
```

### `sites` — 站点导航

列表形式展示，右箭头悬停动画。

| 字段 | 必填 | 说明 |
|------|------|------|
| `name` | **是** | 站点名称 |
| `desc` | **是** | 站点描述 |
| `link` | **是** | 站点 URL，新标签页打开 |
| `icon` | 否 | Emoji / 图片 URL / 留空 |
| `iconSvg` | 否 | SVG 字符串，优先于 `icon` |
| `tag` | 否 | 右侧标签文字 |

```js
{ type: 'sites', title: '站点导航', items: [
  { name: '博客', desc: '技术文章', icon: '📝', link: 'https://blog.example.com', tag: 'Blog' }
] }
```

### `tech` — 技术栈

`items` 为字符串数组，显示为圆角标签云。

```js
{ type: 'tech', title: '技术栈', items: ['React', 'TypeScript', 'Rust', 'Docker'] }
```

### `custom` — 自定义

可插入任意 HTML。

| 字段 | 必填 | 说明 |
|------|------|------|
| `html` | **是** | 任意 HTML 字符串 |

```js
{ type: 'custom', title: '关于', html: '<p style="text-align:center">自定义内容</p>' }
```
